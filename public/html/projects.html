SF-Sense Hybrid Native Mobile App

SF-Sense is built on the MEAN stack. The app makes routine api calls to data.sfgov.org to pull daily crime data and stores it in our MongoDB. We use Google's Map API to display crimes and allow users to search by street addresses. The front-end mobile app javascript, using angular and the ionic framework we're able to build a hybrid native mobile app.
